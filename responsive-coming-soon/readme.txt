== Installation ==

1. Upload the entire `responsive-coming-soon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Responsive Coming Soon Panels menu to customize this plugin.
4. Finally Enjoy it.

== Frequently Asked Questions ==

= coming soon page is not displayed? =

Check that you coming soon mode is enabled from setting, if coming soon is not enable then enabled it.  Now open a new browser where you are not logged in.  Open your site url in this new browser.

= coming soon enabled but i can't see the coming soon page =

If you are using any cache plugin then first deactivate it now check your site in different browser.
